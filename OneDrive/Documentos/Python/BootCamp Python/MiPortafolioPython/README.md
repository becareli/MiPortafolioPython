# Modelado y Gestión de Base de Datos Relacional - Portafolio AECO

Este módulo presenta el diseño y la implementación de una base de datos relacional centrada en la gestión de clientes, proyectos y diagnósticos energéticos en el sector arquitectura y construcción. Refleja mi enfoque estratégico para digitalizar procesos y potenciar la eficiencia de organizaciones AECO.

## Estructura principal

- **clientes:** Registro de datos relevantes (nombre, contacto, dirección) de cada cliente.
- **proyectos:** Descripción, fechas y vinculación de proyectos con los clientes correspondientes.
- **diagnosticos:** Resultados y observaciones de evaluaciones energéticas asociadas a cada proyecto.

## Propósito profesional

Como arquitecto y emprendedor, creo que el dominio en bases de datos y la normalización de la información son clave para construir soluciones tecnológicas escalables y confiables. Este repositorio evidencia mi capacidad para articular proyectos reales integrando gestión de datos, análisis y reporting para la toma de decisiones.

## Ejecución de consultas

1. Selecciona la base de datos:
   ```
   USE portafolio_db;
   ```
2. Ejecuta los scripts de creación y carga provistos (`creacion_tablas.sql`, `insert_datos.sql`, etc).
3. Utiliza las queries incluidas para explorar y analizar la información.
